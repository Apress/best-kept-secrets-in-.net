This solution (CRM.sln) contains both the VB and C# example code 
associated with the book "Best Kept Secrets in .NET" by Deborah Kurata.

To run the VB version of the example, right-click on the CRMWin project and select Set as Startup Project.
To run the C# version of the example, right-click on the CRWin_CSharp project and select Set as Startup Project.

NOTE: This solution requires the Microsoft Application Data Application blocks. The blocks can be downloaded
from the Microsoft site: msdn.microsoft.com/patterns

NOTE: The CRMBOTester project requires NUnit, which can be downloaded from the site: http://sourceforge.net/projects/nunit
